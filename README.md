# Python_Ludo_Dice_Roller
A simple GUI-based Dice Rolling Game built using Python and Tkinter. This project simulates dice rolls with an interactive interface, making it a beginner-friendly implementation of game logic and GUI programming.


# 🎲 Dice Rolling Game (Tkinter)

## 📌 Overview

This project is a **GUI-based Dice Rolling Game** developed using **Python (Tkinter)**. It simulates a dice roll and displays the result visually. The project is designed to demonstrate basic concepts of GUI development, random number generation, and event handling in Python.

---

## 🚀 Features

* 🎲 Random dice rolling functionality
* 🖼️ Graphical User Interface using Tkinter
* 🧠 Simple and interactive gameplay
* 📷 Dice images display based on outcome
* 🖱️ Button-based user interaction

---

## 🛠️ Technologies Used

* Python
* Tkinter (GUI)
* PIL (Python Imaging Library)
* Random module

## 🎮 How It Works

* Click the **Roll Dice** button
* The program generates a random number (1–6)
* The corresponding dice image is displayed
* The result updates dynamically on the GUI
